Curation Engine

One poster - Gysin and Burroughs' RUB OUT THE WORD - taken apart into its parts. Each part is one p5.gysin verb, shown on its own with a live preview and a block you can copy. At the end they assemble into the whole sheet. The page reads as one piece because every layer speaks the same material language: wobble, dropout, rubout, ink.

01 · grid()

The frame

The modular lattice that holds everything. Heavy, ink-bled black rules divide the sheet into cells you can place fields into.

start

Divide the sheet

Decide how many columns and rows the page is built from.

build

Draw ink-bled rules

The frame carries bleed, so the black lines build up unevenly like a worn print.

result

Cells to place into

grid() returns each cell rectangle, so later layers land declaratively.

// a modular, ink-bled frame; returns the cell rectangles
const cells = plot.grid(60, 60, 680, 520, 4, 3, {
  frame: { bleed: 0.3, wobble: 0.7, strokeWeight: 2 }
});
// cells[i] = { x, y, w, h, col, row }

02 · permute() + textCutup()

The permutation block

The engine of the piece: reorder the words, then slice each line as letter contours. Language and image stay separate - one addon reorders, the core cuts up.

start

Reorder the words

GysinText.permute() returns plain strings, the source line first.

build

Cut up each line

textCutup() slices every line into horizontal bands and offsets them.

result

Legible drifting to torn

Raise the slice offset per line and the poem walks from readable to ruptured.

const rows = GysinText.permute("RUB OUT THE WORD", {
  seed: 1961, limit: 5, order: "walk"
});
rows.forEach((row, i) => {
  plot.textCutup(row, 40, 60 + i * 60, {
    size: 26, slices: 6 + i, sliceOffset: 2 + i * 2
  });
});

03 · letters()

The letter field

Where words dissolve. A box is filled with letters drawn from the phrase, each glyph independently varied, so meaning breaks down into pure letterform.

start

Give it a phrase

The source string becomes the pool of letters the field is built from.

build

Fill a box, jitter each glyph

glyphJitter gives every letter its own small deviation - no two alike.

result

A decaying letter mist

Add dropout and the field thins toward noise at its edges.

plot.letters("RUBOUTTHEWORD", 60, 60, 680, 520, {
  size: 16,
  glyphJitter: 0.7,   // every letter varies
  wobble: 0.5,
  dropout: 0.05       // the field thins into noise
});
04 · symbols()

The symbol band

A texture of typewriter operators - the rhythmic noise band under the words. Same idea as the letter field, but the pool is punctuation instead of letters.

start

Pick an operator set

A short string of marks becomes the texture's alphabet.

build

Cluster them into runs

cluster repeats marks in short bursts, so the band reads as rhythm, not static.

result

A rational noise strip

The band grounds the expressive layers with a machined, even texture.

plot.symbols(60, 120, 680, 300, {
  set: '/"()-:',   // the built-in alphabet's marks
  cluster: true,
  size: 16
});

Operator glyphs like + # & are not in the built-in vector alphabet. For those, type them with the underwood() verb (its Hershey face has them) or pass an outline font.

05 · asemic()

The asemic gesture

Writing with no letters. A smooth turning walk that drifts back toward its centre reads as a compact, cursive scribble - the calligraphic tangle that buries the words.

start

Give it a box

The gesture fills the rectangle you hand it, one tangle per cell.

build

A turning, centre-seeking walk

The path loops and drifts home, so it looks hand-written, not like an angular net.

result

Script without language

A grid of these reads as rows of writing you can feel but not read.

// one cursive gesture per cell of a small grid
for (let r = 0; r < 2; r++) {
  for (let c = 0; c < 4; c++) {
    plot.asemic(60 + c * 170, 60 + r * 240, 150, 210, {
      wobble: 1.4
    });
  }
}
06 · underwood()

The typewriter

The typed voice, period-correct. Single-stroke letters at a fixed pitch, named after the Underwood No. 5 that Burroughs cut up the world on. No bold, no italic - "bold" is just the same key struck twice.

start

Type the line

Defaults alone give a period page: fixed pitch, 6 lines-per-inch, light strike wear.

build

Only period decoration

underline, bold: true (double-strike), spacing, and typed dividers.

result

A real typed sheet

Every letter is one pen line, so it plots exactly as drawn.

// needs p5.gysin.underwood.min.js loaded
plot.underwood("RUB OUT THE WORD", 60, 90, {
  size: 30, underline: 1
});
plot.underwood("DOUBLE-STRUCK", 60, 170, { bold: true });
plot.underwood("* * * * * * * *", 60, 230);  // a typed divider

07 · assembly

One sheet, every part

The verbs stack into the whole poster: a frame holds a typed permutation block over a dissolving letter field, an operator band, and an asemic gesture at the foot. One seed, one material vocabulary, one page.

const plot = new GysinPlot({ seed: 1961, width: 560, height: 760 });

// frame -> typed block -> letter field -> operator band -> asemic
plot.grid(30, 30, 500, 700, 1, 1, { frame: { bleed: 0.3, wobble: 0.6 } });

const rows = GysinText.permute("RUB OUT THE WORD", { seed: 1961, limit: 5 });
plot.underwood(rows.join("\n"), 60, 90, { size: 15 });

plot.letters("RUBOUTTHEWORD", 60, 330, 440, 150, { size: 13, glyphJitter: 0.7, dropout: 0.05 });
plot.underwood('+  "  #  &  /   +  "  #  &  /', 60, 520, { size: 15, wear: 0.6 });
plot.asemic(60, 560, 440, 150, { wobble: 1.6 });

plot.draw();
