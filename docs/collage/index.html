Collage

Collage, here, is done to language, not to paper. Words are the scraps: a phrase is cut apart, its words change neighbours, and the new order is fixed to the sheet as drawn lines. Three acts - cut, arrange, glue - and one word, COLLAGE, run through every function of the library, each with a live preview and a block you can copy. At the end the acts fix themselves to one sheet.

01 · grid()

The sheet

Every collage starts with the board it is glued to. grid() divides the sheet with ink-bled rules and hands back the cell rectangles, so every later fragment knows where it may land.

cut

Divide the sheet

Choose the columns and rows the board is built from.

arrange

Ink-bled rules

The frame carries bleed, so the black lines build up unevenly like a worn print.

glue

Cells to place into

grid() returns each cell, so fragments land declaratively.

// the board: a modular, ink-bled frame
const cells = plot.grid(60, 60, 680, 520, 4, 3, {
  frame: {
    bleed: 0.3,
    breathe: 0.7,
    strokeWeight: 2
  }
});
// cells[i] = { x, y, w, h, col, row }
02 · line() · rect() · circle() · path()

Scraps

The non-text material: strips, patches, rings, and free clippings. Four primitives with torn edges - breathe makes them hand-cut, dropout tears gaps, repeat and overshoot let the scissors pass twice and run past the corner.

cut

Four kinds of scrap

A strip, a patch, a ring, and a free polyline clipping.

arrange

Torn, not drawn

breathe and dropout give every edge the uncertainty of a hand tear.

glue

Passes of the blade

repeat doubles the pass, overshoot runs past the corner like a real cut.

plot.line(60, 80, 680, 96, {
  breathe: 1.2,
  repeat: 2
});
plot.rect(60, 140, 260, 180, {
  breathe: 0.8,
  overshoot: 6
});
plot.circle(500, 230, 150, {
  breathe: 0.9,
  dropout: 0.08
});
plot.path([[380, 420], [520, 380], [660, 440], [600, 520]], {
  breathe: 1.0
});
03 · text() · breathe · dropout · rubout

The word

The word is the material - not what is printed on the scrap, but the scrap itself. text() traces it as vector contours; the material options set its condition: breathe is the tremor of the hand, dropout the print that skips, rubout the eraser that has already been here.

cut

One clean clipping

The word traced as letter contours, nearly steady.

arrange

Its condition

breathe, dropout, and rubout age the same clipping in states.

glue

Same trace, worn

Each state is the same paths, sampled with more uncertainty.

plot.text("COLLAGE", 60, 120, {
  size: 64,
  breathe: 0.4
});
plot.text("COLLAGE", 60, 260, {
  size: 64,
  breathe: 1.4,
  dropout: 0.08
});
plot.text("COLLAGE", 60, 400, {
  size: 64,
  breathe: 2.2,
  dropout: 0.2,
  rubout: 0.2
});
04 · textCutup()

Scissors

The defining act. textCutup() slices the word into horizontal bands and shifts each band sideways, exactly like scissors through a printed line. Cut shallow and it stays legible; cut deep and the word becomes texture.

cut

Slice into bands

slices sets how many strips the scissors make.

arrange

Shift each band

sliceOffset slides the strips out of register.

glue

Two depths of cut

The same word twice: one careful cut, one deep cut with tears.

plot.textCutup("COLLAGE", 60, 180, {
  size: 72,
  slices: 6,
  sliceOffset: 8
});
plot.textCutup("COLLAGE", 60, 380, {
  size: 72,
  slices: 10,
  sliceOffset: 24,
  dropout: 0.08,
  rubout: 0.1
});
05 · GysinText.permute() + textCutup()

Arrange

This is the essence of the whole page: collage as an act on word order. The three acts themselves - cut, arrange, glue - are handed to GysinText.permute(), every word changes neighbours, and every new order runs through the scissors again. The addon reorders language; the core cuts the image.

cut

Loose words

permute() returns plain strings, the source line first.

arrange

Every order

Orders walk, random, lexical, rotate; duplicates yield only unique lines.

glue

Cut each new order

Deeper offsets per line walk the poem from readable to ruptured.

const lines = GysinText.permute("CUT ARRANGE GLUE", {
  seed: 1919,
  limit: 5,
  order: "walk"
});

lines.forEach((line, index) => {
  plot.textCutup(line, 60, 90 + index * 90, {
    size: 34,
    slices: 5 + index,
    sliceOffset: 2 + index * 4
  });
});
06 · letters() · glyphJitter

The letter box

Cut the word small enough and only its letters remain: a box of loose type, like clippings sifted from a magazine. letters() fills a region from the word's own letter pool, and glyphJitter gives every glyph its own small deviation - no two clippings alike.

cut

Down to letters

The word becomes the pool the field is built from.

arrange

Fill a region

Rows of loose type fill the box you hand it.

glue

Each glyph unique

glyphJitter varies every letter; dropout thins the field into noise.

plot.letters("COLLAGE", 60, 60, 680, 520, {
  size: 16,
  glyphJitter: 0.7,
  breathe: 0.5,
  dropout: 0.05
});
07 · direction · the turned sheet

The turn

Writing has a direction, and the sheet can be turned against it. The typed line runs left to right; the word answers in columns, top to bottom, written straight over what was already there. Nothing here is a new function: it is text() released from its row, one letter under the last, direction against direction until the page starts to weave.

cut

One letter at a time

A column is the word freed from its row: each letter set under the last.

arrange

Turn the sheet

Write across what is already written; the row keeps running underneath.

glue

The weave

Where the directions cross, the ink thickens; the words stay, the reading dissolves.

// the row: a typed line, left to right
plot.underwood("cut arrange glue cut arrange glue", 60, 320, {
  size: 22
});

// the columns: the word written downwards, over the row
const word = "COLLAGE";
for (let column = 0; column < 4; column++) {
  for (let i = 0; i < word.length; i++) {
    plot.text(word[i], 90 + column * 170, 80 + i * 68, {
      size: 44,
      breathe: 0.9,
      bleed: 0.2
    });
  }
}
08 · symbols()

The mark box

Next to the letters, a box of punctuation: the small hard marks that hold a page together. symbols() works like the letter box but pools marks instead of letters, and cluster repeats them in short bursts so the band reads as rhythm, not static.

cut

Pick an alphabet of marks

A short string of punctuation becomes the texture's pool.

arrange

Cluster into runs

Bursts of the same mark read as machine rhythm.

glue

A rational noise strip

The band grounds the expressive fragments with an even texture.

plot.symbols(60, 120, 680, 300, {
  set: '/"()-:',
  cluster: true,
  size: 16
});

Marks like + # & are not in the built-in vector alphabet. Type those with underwood() (its face has them) or pass an outline font.

09 · underwood()

The typed label

Every collage carries a typed strip: the caption pasted under the image. underwood() types it period-correct - single-stroke letters at a fixed pitch, no bold, no italic. "Bold" is the same key struck twice; the underline is the underscore ladder. And the machine has two voices: capitals shout the word, lowercase writes the note underneath.

cut

Type the strip

Defaults give a period page: fixed pitch, six lines per inch, light wear.

arrange

Only period decoration

underline, bold: true (double-strike), and typed dividers.

glue

One pen line per letter

Single-stroke, so it plots exactly as typed.

// needs p5.gysin.underwood.min.js loaded
plot.underwood("COLLAGE", 60, 90, {
  size: 30,
  underline: 1
});
plot.underwood("CUT ARRANGE GLUE", 60, 170, {
  bold: true
});
plot.underwood("cut, arrange, glue. the small voice.", 60, 230);
plot.underwood("* * * * * * * *", 60, 290);

10 · fill: "hatch" · fill: "cross"

Tone blocks

Collage needs dark paper too: blocks of tone to paste behind and between the words. A plotter cannot pour ink, so fill builds tone from strokes - hatch lays parallel lines, cross lays two perpendicular hatchings. The patch and its filling are one family: the outline is the hand-drawn rect(), and every hatch stroke is the same free line - it breathes, runs past the edge, skips, and soaks. Craft over perfection: the mistakes are the tone.

cut

A patch of tone

Any closed shape can carry a fill; counters stay open.

arrange

Set the weave

hatchSpacing sets the density, hatchAngle turns it; cross lays stroke over stroke in two directions, writing woven over writing.

glue

Let the hand show

breathe, overshoot, dropout, and bleed reach the fill strokes too; stats() counts them as fillPaths.

plot.rect(60, 80, 280, 200, {
  fill: "hatch",
  hatchSpacing: 5,
  hatchAngle: 30,
  breathe: 0.9,
  overshoot: 5,
  bleed: 0.25
});
plot.rect(420, 80, 280, 200, {
  fill: "cross",
  hatchSpacing: 7,
  hatchAngle: 15,
  breathe: 1.2,
  dropout: 0.06,
  overshoot: 4
});
plot.circle(380, 430, 180, {
  fill: "hatch",
  hatchSpacing: 4.5,
  hatchAngle: 75,
  breathe: 0.9,
  dropout: 0.05,
  bleed: 0.35
});

11 · bleed · repeat

Glue & ink

Glue, in a collage of words, is commitment: the pass that fixes the new order to the sheet. bleed is that commitment made visible - extra passes that thicken a trace unevenly, ink building up where the sheet was worked longest. It is selective: you decide which words soak.

cut

One dry strip

A plain line is a clean, single pass.

arrange

Let it soak

bleed adds uneven passes; bleedPasses caps how many.

glue

Measured, not guessed

stats() reports bleedPaths and overdrawRatio, so build-up stays plottable.

// dry strip
plot.line(60, 120, 680, 130, {
  breathe: 0.8
});
// soaked strips: ink builds up in passes
plot.line(60, 240, 680, 255, {
  breathe: 0.8,
  bleed: 0.5
});
plot.line(60, 360, 680, 380, {
  breathe: 0.8,
  bleed: 0.9,
  bleedPasses: 3
});
12 · rub()

Wear

A fragment that is handled wears out. rub() is the intent verb for that whole gesture: the word appears legible, returns cut up, and is finally buried under an asemic scribble - three copies, one call, one decay knob to scale the wear.

cut

One call

rub() composes text, textCutup, and asemic with strong defaults.

arrange

One knob

decay scales breathe, drift, dropout, and rubout across every copy at once.

glue

Escape hatches stay

tail, stages, and tangles replace the recipe when you need to.

// the whole gesture, defaults and all
plot.rub("COLLAGE", 60, 220, {
  size: 56
});

13 · asemic()

The margin note

In the margin of every collage: handwriting that is not language. asemic() draws a smooth turning walk that drifts back toward its centre - a compact cursive scribble you can feel but not read. Rows of them read as the note the maker left.

cut

Give it a box

The gesture fills the rectangle you hand it, one tangle per cell.

arrange

A centre-seeking walk

The path loops and drifts home, so it reads as writing, not as a net.

glue

Script without language

A row of tangles becomes a line of unreadable annotation.

// one cursive gesture per cell of a small grid
for (let r = 0; r < 2; r++) {
  for (let c = 0; c < 4; c++) {
    plot.asemic(60 + c * 170, 60 + r * 240, 150, 210, {
      breathe: 1.4
    });
  }
}
14 · layer · penMap · export

Pens & off the table

The finished collage leaves the table. Give every material its own layer, map layers to physical pens, and the same captured paths go out as SVG for the plotter driver, JSON for data, or HPGL for the classics. Nothing is redrawn: export is the drawing.

cut

Name the materials

Each call carries a layer: frame, word, tone.

arrange

Map layers to pens

penMap assigns each layer a physical pen number.

glue

Ship the sheet

downloadSVG, downloadJSON, downloadHPGL; stats() predicts the plot.

plot.rect(60, 60, 680, 520, {
  layer: "frame",
  breathe: 0.9,
  overshoot: 6,
  bleed: 0.25
});
plot.text("COLLAGE", 120, 320, {
  size: 72,
  breathe: 0.8,
  layer: "word"
});
plot.rect(120, 380, 240, 120, {
  fill: "hatch",
  hatchSpacing: 5,
  hatchAngle: 30,
  breathe: 1.0,
  dropout: 0.05,
  layer: "tone"
});

const page = { format: "A4", margin: 18 };
plot.downloadSVG("collage.svg", {
  page,
  optimize: true,
  penMap: { frame: 1, word: 2, tone: 3 }
});
console.log(plot.stats({ page }));

15 · assembly

The collage

Stamped, written, turned, overwritten, erased, spattered. The incised table rolls over the sheet twice and lands off register; the typed strip runs left to right and the word answers top to bottom; fields collide instead of stacking; the worn word erases what it crosses, and flecks of ink let chance keep the last word. One seed, every act, no hierarchy.

const plot = new GysinPlot({ seed: 1919, width: 560, height: 760 });

// the roller: an incised table, stamped twice, off register.
// The first pass soaks; the second is one clean stroke of the pen.
plot.grid(30, 30, 500, 700, 3, 4, { frame: { bleed: 0.4, breathe: 0.9 } });
plot.grid(38, 24, 500, 700, 3, 4, { frame: { breathe: 1.6 } });

// left to right: the typed permutation strip
const lines = GysinText.permute("CUT ARRANGE GLUE", { seed: 1919, limit: 4 });
plot.underwood(lines.join("\n"), 56, 120, { size: 14, wear: 0.5 });

// top to bottom: the word turned, written over the strip.
// One pass or two: the middle column is struck twice.
const word = "COLLAGE";
for (let column = 0; column < 3; column++) {
  for (let i = 0; i < word.length; i++) {
    plot.text(word[i], 120 + column * 150, 90 + i * 86, {
      size: 38,
      breathe: 1.0,
      bleed: 0.2,
      repeat: column === 1 ? 2 : 1
    });
  }
}

// fields collide: sifted letters across the marks, tone behind
plot.letters("COLLAGE", 56, 430, 440, 150, { size: 13, glyphJitter: 0.8, dropout: 0.02 });
plot.symbols(200, 500, 300, 90, { set: '/"()-:', cluster: true, size: 13 });
plot.rect(330, 560, 170, 120, { fill: "hatch", hatchSpacing: 5, hatchAngle: 30, breathe: 1.0, repeat: 2 });

// erased and overwritten: the worn word drags across the field
plot.rub("COLLAGE", 90, 620, { size: 34, decay: 1.6 });
plot.asemic(300, 590, 200, 110, { breathe: 1.8 });

// spatter: chance keeps the last word
for (let i = 0; i < 40; i++) {
  const angle = i * 2.399;
  const radius = 10 * Math.sqrt(i);
  plot.line(300 + Math.cos(angle) * radius, 380 + Math.sin(angle) * radius * 1.5,
            302 + Math.cos(angle) * radius, 382 + Math.sin(angle) * radius * 1.5, { breathe: 3 });
}

plot.draw();
