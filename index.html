Enter the three studies
Three studies · one movement · v0.2.0

Cut. Permute. Free.

The opening is not a catalogue of effects. Three studies perform three operations on the same material: montage, repetition with difference, and release into gesture. Every preview is a live sketch; the snippets show the transferable idea, not every line.

01 · CUT · textCutup()

Language is material

Slice text into unstable bands, then combine dropout, rubout, repeat, and drift.

// the same line, sliced twice - the second cut deeper
plot.textCutup("I THINK THEREFORE I AM", 76, 140, {
  size: 42,
  slices: 6,
  sliceOffset: 8
});
plot.textCutup("I THINK THEREFORE I AM", 76, 260, {
  size: 42,
  slices: 9,
  sliceOffset: 22,
  dropout: 0.09,
  rubout: 0.12
});

02 · PERMUTE · GysinText.permute()

Chance is a method

Reorder the words first, then send every resulting line through the original contour cut-up and plotter pipeline.

const lines = GysinText.permute("I LOVE YOU", {
  seed: 1960,
  limit: 6,
  order: "walk"
});

lines.forEach((line, index) => {
  plot.textCutup(line, 48, 90 + index * 46, {
    slices: 7 + index,
    sliceOffset: 12 + index * 4
  });
});

03 · FREE · rub()

Writing escapes the word

One intent verb. rub() composes text, cut-up, and asemic into a single decaying gesture - the whole piece in one call, defaults and all.

const plot = new GysinPlot({ seed: 1960 });

// one word, worn away - legible → cut up → asemic, in one call
plot.rub("WORN", 46, 72, {
  size: 48
});

plot.draw();

underwood() · single-stroke · period

Typewriter

Burroughs' Underwood No. 5 as an intent verb. Single-stroke letters, period pitch and line height, no bold or italic - "bold" is just the same key struck twice.

// defaults alone = a period-correct typed page
plot.underwood("KICK THAT HABIT MAN", 60, 90);

// the only decoration the machine had:
plot.underwood("HEADING", 60, 130, { underline: 1 });
plot.underwood("BOLD?", 60, 160, { bold: true });   // struck twice

path() · no text

Signal Score

Treat paths as fragile signals: no typography, only a scored machine reading made from vector paths and markers.

const signal = [];
for (let x = 0; x <= 620; x += 8) {
  signal.push([x, y + sin(x * 0.04) * 28]);
}

// row after row of the same signal, each a step more worn
plot.path(signal, {
  dropout: 0.05 + row * 0.01,
  repeat: 2,
  rubout: 0.06 + row * 0.03
});

Draw as traces

Not pixels. Paths.

Every call captures a vector shape, samples it into points, applies seeded damage and hesitation, then draws or exports the same generated polylines.

Capture

Use plot.line(), plot.rect(), plot.circle(), plot.polygon(), plot.path(), plot.text(), and plot.textCutup().

Disturb

Shape the trace with wobble, dropout, repeat, drift, rubout, fray, overshoot, and pressure.

Export

Keep output plotter-friendly with SVG, JSON, and HPGL export. No raster erase, no canvas pixel analysis.
