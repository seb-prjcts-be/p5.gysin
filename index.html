Enter the three studies

Cut. Permute. Free.

The opening is not a catalogue of effects. Three studies perform three operations on the same material: montage, repetition with difference, and release into gesture. Every preview is a live sketch; the snippets show the transferable idea, not every line.

01 · CUT · textCutup()

Language is material

Slice text into unstable bands, then combine dropout, rubout, repeat, and drift.

// the same line, sliced twice - the second cut deeper
plot.textCutup("I THINK THEREFORE I AM", 76, 140, {
  size: 42,
  slices: 6,
  sliceOffset: 8
});
plot.textCutup("I THINK THEREFORE I AM", 76, 260, {
  size: 42,
  slices: 9,
  sliceOffset: 22,
  dropout: 0.09,
  rubout: 0.12
});

02 · PERMUTE · chant()

Coincidence is a method

One call reorders the words and sends every resulting line through the contour cut-up, each cut a little deeper. The library does the orchestration.

// the whole permutation poem in one call: the phrase
// through its orders, every line cut a little deeper,
// through real outline letterforms
plot.chant("I LOVE YOU", 48, 90, {
  lines: 6,

03 · FREE · rub()

Writing escapes the word

One intent verb. rub() composes text, cut-up, and asemic into a single decaying gesture - the whole piece in one call, defaults and all.

const plot = new GysinPlot({ seed: 1960 });
const outlineFont = await loadFont(FONT_URL);   // fills the legible head

// one word, worn away - legible → cut up → asemic, in one call
plot.rub("WORN", 46, 72, {
  size: 48,
  font: outlineFont
});

plot.draw();

text() · one word, three states

First Trace

Start with clean vector forms and let p5.gysin turn them into uncertain plotter traces.

const plot = new GysinPlot({ seed: 1960 });
const outlineFont = await loadFont(FONT_URL);   // real letterforms

// the same word, three states of the same trace
plot.text("FIRST TRACE", 68, 90, {
  font: outlineFont,
  size: 54,
  breathe: 0.4
});
plot.text("FIRST TRACE", 68, 180, {
  font: outlineFont,
  size: 54,
  breathe: 1.4,
  dropout: 0.08
});
plot.text("FIRST TRACE", 68, 270, {
  font: outlineFont,
  size: 54,
  breathe: 2.2,
  dropout: 0.2,
  rubout: 0.2
});

underwood() · single-stroke · period

Typewriter

Burroughs' Underwood No. 5 as an intent verb. Single-stroke letters, period pitch and line height, no bold or italic - "bold" is just the same key struck twice.

// defaults alone = a period-correct typed page
plot.underwood("KICK THAT HABIT MAN", 60, 90);

// the only decoration the machine had:
plot.underwood("HEADING", 60, 130, { underline: 1 });
plot.underwood("BOLD?", 60, 160, { bold: true });   // struck twice

breathe · dropout · repeat · rubout

Parameter Lab

One slider owns one element, each in its own ink: breathe bends the whole title, rubout tears the word, repeat thickens the band, dropout opens the ruled block.

// one slider owns one element, each in its own ink
plot.text("RUB OUT", 42, 90, {
  size: 44,
  breathe: 1.4,
  stroke: "#b26a00"
});
plot.text("ERASE", 42, 160, {
  size: 30,
  rubout: 0.12,
  stroke: "#b0203a"
});
plot.path(wave, {   // wave: a sampled sine row
  repeat: 2,
  drift: 2.4,
  stroke: "#2b4fb0"
});
for (let i = 0; i < 6; i++) {   // a ruled block, no fill
  plot.line(42, 240 + i * 9, 380, 240 + i * 9, {
    dropout: 0.16,
    stroke: "#0f7a6c"
  });
}

path() · no text

Signal Score

Treat paths as fragile signals: no typography, only a scored machine reading made from vector paths and markers.

const signal = [];
for (let x = 0; x <= 620; x += 8) {
  signal.push([x, y + sin(x * 0.04) * 28]);
}

// row after row of the same signal, each a step more worn
plot.path(signal, {
  dropout: 0.05 + row * 0.01,
  repeat: 2,
  rubout: 0.06 + row * 0.03
});

bleed · the additive counterpart

Ink Bleed

dropout and rubout take line material away; bleed gives chosen fragments extra, slightly shifted passes - ink gathering where the pen returned. No cut-up: the phrase stays whole. tool: "blade" strips the build-up again for cutting machines.

// the same whole line three times - the ink returns harder each
// row, gathering along real outline letterforms
plot.text("I AM THAT I AM", 42, 90, { font: outlineFont, size: 34 });
plot.text("I AM THAT I AM", 42, 180, {
  font: outlineFont,
  size: 34,
  bleed: 0.22,
  bleedPasses: 2
});
plot.text("I AM THAT I AM", 42, 270, {
  font: outlineFont,
  size: 34,
  bleed: 0.4,
  bleedPasses: 3,
  bleedSpread: 1.2
});

Draw as traces

Not pixels. Paths.

Every call captures a vector shape, samples it into points, applies seeded damage and hesitation, then draws or exports the same generated polylines.

Capture

Use plot.line(), plot.rect(), plot.circle(), plot.polygon(), plot.path(), plot.text(), and plot.textCutup().

Disturb

Shape the trace with breathe, dropout, repeat, drift, rubout, fray, overshoot, and pressure.

Export

Keep output plotter-friendly with SVG, JSON, and HPGL export. No raster erase, no canvas pixel analysis.
