line() · text() · circle()

First Trace

Start with clean vector forms and let p5.gysin turn them into uncertain plotter traces.

const plot = new GysinPlot({ seed: 1960 });

plot.line(70, 160, 450, 178, {
  wobble: 1.2,
  dropout: 0.04,
  overshoot: 8
});

plot.text("FIRST TRACE", 68, 255, {
  size: 54,
  wobble: 1.4,
  dropout: 0.08,
  repeat: 2
});

GysinText.permute() · textCutup()

Permutation Poem

Reorder the words first, then send every resulting line through the original contour cut-up and plotter pipeline.

const regels = GysinText.permute("I LOVE YOU", {
  seed: 1960,
  limit: 6,
  order: "walk"
});

regels.forEach((regel, index) => {
  plot.textCutup(regel, 48, 90 + index * 46, {
    slices: 7 + index,
    sliceOffset: 12 + index * 4
  });
});

Signal Score

Treat paths as fragile signals: no typography, only a scored machine reading made from vector paths and markers.

const signal = [];
for (let x = 0; x <= 620; x += 8) {
  signal.push([x, y + sin(x * 0.04) * 28]);
}

plot.path(signal, {
  dropout: 0.07,
  repeat: 2,
  rubout: 0.16

Draw as traces

Not pixels. Paths.

Every call captures a vector shape, samples it into points, applies seeded damage and hesitation, then draws or exports the same generated polylines.

Capture

Use plot.line(), plot.rect(), plot.circle(), plot.polygon(), plot.path(), plot.text(), and plot.textCutup().

Disturb

Shape the trace with wobble, dropout, repeat, drift, rubout, fray, overshoot, and pressure.

Export

Keep output plotter-friendly with SVG, JSON, and HPGL export. No raster erase, no canvas pixel analysis.
