p5.gysin

Vector-first traces for cut-up text, rubout gaps, uncertain contours, and plotter-friendly export.

Read the guide
Draw as traces

Not pixels. Paths.

Every call captures a vector shape, samples it into points, applies seeded damage and hesitation, then draws or exports the same generated polylines.

Capture

Use `plot.line()`, `plot.rect()`, `plot.circle()`, `plot.polygon()`, `plot.path()`, `plot.text()`, and `plot.textCutup()`.

Disturb

Shape the trace with `wobble`, `dropout`, `repeat`, `drift`, `rubout`, `fray`, `overshoot`, and `pressure`.

Export

Keep output plotter-friendly with SVG, JSON, and HPGL export. No raster erase, no canvas pixel analysis.

Get Started

<script src="https://cdn.jsdelivr.net/npm/p5@2/lib/p5.js"></script> <script src="p5.gysin.min.js"></script> function setup() { createCanvas(800, 800); const plot = new GysinPlot({ seed: 1960 }); plot.textCutup("RUB OUT THE WORD", 80, 180, { size: 72, wobble: 2, dropout: 0.12, repeat: 2, rubout: 0.25 }); plot.draw(); }
