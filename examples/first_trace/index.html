First Trace

One word, one gesture, drawn three times down a single faint backbone line — from a solid, cross-hatch-filled head to a hollow, broken tail. Read it top to bottom. Each reseed steps one word further along a five-word sentence (FIRST TRACE → SLOW HAND → LOST WORD → OPEN MARK → GONE SOON); the seed and decay printed on the sheet reproduce any variation.

  • legible — the word cross-hatch filled: solid, ink-heavy head
  • cut-up — sliced and drifting: outline only, coming apart
  • asemic — buried under a scribble: the hollow, near-empty tail

The decay knob (Decay +/− or the ↑/↓ arrow keys) scales wobble, dropout and rubout across the whole trace at once, from 0× fully clean to 2.5× fully wiped.

Core snippet
// The whole library, in three lines — this alone is a complete sketch:
const plot = new GysinPlot({ seed: 1960 });
plot.text("FIRST TRACE", 46, 248);   // clean, mechanical text
plot.draw();

// Everything below is OPTIONAL disturbance layered on top. Defaults are all
// zero, so options are opt-in. This example adds, in order:

// 1) one "decay" knob scaling wobble/drift/dropout/rubout/fray on any block
const decayed = (o) => ({ ...o,
  wobble: (o.wobble ?? 0) * decay, dropout: (o.dropout ?? 0) * decay,
  rubout: (o.rubout ?? 0) * decay, fray: (o.fray ?? 0) * decay });

// 2) the same word drawn three times, decaying from a solid, cross-hatch
//    filled head to a hollow, broken tail — one pass over a stage table
for (const stage of STAGES) {
  const opts = decayed({ size: 46, ...stage.extra, rubout: stage.rubout });
  if (font && stage.fill) opts.fill = stage.fill;   // only the head is filled
  plot[stage.verb](word, stage.x, stage.y, opts);   // text / textCutup / text
}

// 3) an asemic scribble that buries the last, ghostly copy of the word
for (const t of TANGLES) {
  plot.asemic(t.x, t.y, t.w, t.h, decayed({ dropout: t.dropout }));
}

plot.draw();
Decay 1.0× · seed 1960 — both printed on the sheet, so an exported SVG records its own setting. Click the canvas or New variation for the next word; Download SVG saves the plotter-ready file.
