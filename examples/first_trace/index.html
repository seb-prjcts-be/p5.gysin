First Trace

A minimal composition that shows the core idea: define clean vector forms, then let p5.gysin redraw them as unstable plotter traces. Click the canvas to reseed the drawing.

Core snippet
const plot = new GysinPlot({ seed: 1960 });

plot.line(70, 160, 450, 178, {
  wobble: 1.2,
  dropout: 0.04,
  overshoot: 8
});

plot.text("FIRST TRACE", 68, 255, {
  size: 54,
  wobble: 1.4,
  dropout: 0.08,
  repeat: 2
});

plot.draw();
Click the canvas to reseed. Press S to download SVG.
