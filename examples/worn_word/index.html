rub()

Worn Word

One word, worn away. rub() is an intent verb: it composes text, textCutup, and asemic into a single decaying gesture - the same word drawn three times, from a solid legible head to an asemic scribble that buries the last, ghostly copy. This whole sketch is one call. It is the First Trace gesture, which builds the same thing by hand in ~40 lines, collapsed into one.

The decay knob (Decay +/− or the ↑/↓ arrow keys) scales the whole disturbance at once, from 0× fully clean to 2.5× fully worn. Every option is optional; the defaults alone draw the full gesture.

Core snippet
// The whole library, in three lines - this alone is a complete sketch:
const plot = new GysinPlot({ seed: 1960 });
plot.rub("WORN", 46, 248);   // legible → cut up → asemic, in one gesture
plot.draw();

// Every option is opt-in. decay scales the whole disturbance from one knob;
// an outline font fills the legible head (without one it stays hollow):
plot.rub("WORN", 46, 248, { decay: 2, font: outlineFont });

// rub() returns the id of every copy and tangle, so each can be frozen,
// rerolled, or exported on its own - like any other p5.gysin shape.
