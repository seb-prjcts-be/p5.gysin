GysinText.permute() → textCutup() → plotter paths

Permutation Poem

The optional text module rearranges the sentence. The original cut-up engine then passes every permutation through repeated type, a symbolic code column, damaged letter bands, and a modular grid. Meaning and letterform are disturbed in two separate passes.

Two independent layers
const regels = GysinText.permute("I LOVE YOU", {
  seed: 1960,
  limit: 6,
  order: "walk"
});

regels.forEach((regel, index) => {
  plot.textCutup(regel, x[index], y[index], {
    font: outlineFont,
    slices: 7 + index,
    sliceOffset: 14 + index * 6,
    dropout: 0.02 + index * 0.01
  });
});
