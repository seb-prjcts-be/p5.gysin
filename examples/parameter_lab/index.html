Parameter Lab

A small control surface for the trace engine. The sliders rebuild the same composition so the effect of each parameter stays readable.

Core snippet
plot.textCutup("RUB OUT", 58, 170, {
  size: 72,
  slices: 7,
  sliceOffset: 18,
  wobble: wobbleValue,
  dropout: dropoutValue,
  repeat: repeatValue,
  rubout: ruboutValue
});
