Parameter Lab

One slider owns one element, in its own colour. Hover a control - or tap a legend row on touch - to preview which element it owns; drag the slider and only that element moves - darkening to a strong figure while the rest fade to a quiet ground, in the canvas and the exported SVG.

  • Wobble bends the title
  • Rubout tears ERASE away
  • Repeat thickens the line band
  • Dropout loosens the hatch bar

The letter field and corner marks obey no slider - fixed reference ink that travels into the SVG. The stats() readout shows total plot cost and the shown element's share of it.

Core snippet
// The whole library, in three lines - this alone is a complete sketch:
const plot = new GysinPlot({ seed: 8319 });
plot.text("RUB OUT", 70, 150);   // clean, mechanical text
plot.draw();

// Everything below is OPTIONAL disturbance layered on top. Defaults are
// all zero, so options are opt-in. This example builds, back to front:

// 1) a plate: thin border + corner registration marks (fixed grey ink)
// 2) the title "RUB OUT", cut up and bent - owned by the WOBBLE slider
plot.textCutup("RUB OUT", 70, 150, { wobble: value("wobble") });

// 3) the word "ERASE", torn away - owned by the RUBOUT slider
plot.textCutup("ERASE", 70, 214, { rubout: value("rubout") });

// 4) a woven line band - each row drawn `repeat` times, owned by REPEAT
plot.path(waveRow(70, 490, y, amp, phase), { repeat: value("repeat") });

// 5) a cross-hatch bar that opens up - owned by the DROPOUT slider
plot.rect(70, 428, 420, 64, { fill: "cross", dropout: value("dropout") });

// 6) a dense letter field: fixed reference ink, obeys no slider
plot.letters("GYSIN", 300, 150, 190, 100, { size: 10 });

plot.draw();

Sweep rubout to tear ERASE from intact to shredded, or repeat to see the woven band thicken and spread wider with every added pass. Hit Reset sliders to return every element to full ink. Use the buttons, or press R (new variation) and S (save SVG).
