The Letter

The typewriter's natural habitat was never the poster - it was the letter. This sheet types one complete piece of correspondence and, along the way, uses every voice the machine ever had: the S p e r r s a t z dateline, the ruled salutation, the double strike, and the row of asterisks that divides a page.

Look closely at the fourth line of events: a word is unsaid the only way the machine allowed - typed over with X's, bold: true striking the same cell twice. And the signature drops to lowercase: both cases live on the same keys.

Core snippet
// Load the underwood module, then the letter is a handful of lines:
const plot = new GysinPlot({ seed: 1912 });
plot.underwood("DEAR READER,", 60, 150, { underline: 1 });
plot.underwood("THE MACHINE HAS ONE FACE, ONE SIZE.\n...", 60, 210);

// unsaying a word, the period way - struck over with X's:
plot.underwood("FORGET", 60, 360);
plot.underwood("XXXXXX", 60, 360, { bold: true });

// and the small voice signs:
plot.underwood("the small hand.", 60, 600);
plot.draw();
New strike retypes the same letter with the next seeded wear - every strike lands differently, the words hold. Download SVG saves the plotter-ready file; every letter is a single pen line. R and S do the same from the keyboard.
