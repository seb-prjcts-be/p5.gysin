SVG · JSON · HPGL

Plotter Export

A compact export workflow for pen plotters and vector tools. The same generated traces are drawn on canvas and exported as SVG, JSON, or HPGL.

Core snippet
const page = {
  width: 210, height: 210, units: "mm",
  margin: 5, scale: 0.375, clip: true
};

const svg = plot.exportSVG({ page, optimize: true });

const json = plot.exportJSON({
  includeGenerated: true
});

const hpgl = plot.exportHPGL({
  page, penMap: { frame: 1, type: 2 }, speed: 20
});

const stats = plot.stats({ page, drawSpeed: 20 });
