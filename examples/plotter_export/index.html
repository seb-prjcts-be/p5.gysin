SVG · JSON · HPGL

Plotter Export

A compact export workflow for pen plotters and vector tools. The same generated traces are drawn on canvas and exported as SVG, JSON, or HPGL. Each layer is stroked in its export-pen colour, so the canvas already is the pen map.

How to read it

  • Pen 1 (ink) — the frame and the registration work: a cross in every corner plus a circle and triangle on the diagonal. The frame hesitates mid-edge and the crosses overshoot and fray, the way a hand-set mark runs past its own centre.
  • Pen 2 (red) — the type: the hero word and the cut-up subtitle, which spells the four export layers FRAME / TYPE / HATCH / REG so the plate captions its own pen map.
  • Pen 3 (blue) — the texture that bridges type into tone: a decaying letter field, a mid-tone accent block, an airy field of operator glyphs, and a dense cross-hatch that anchors the plate in solid, dark tone.
  • Legend — names which pen draws which layer, its path count, and roughly how many seconds it keeps the machine busy (drawn length ÷ draw speed). The status line adds the total paths, millimetres of ink, and estimated plot time.

Controls

  • New variation — or press R — rerolls the plate: a new symbol set, a re-angled cross-hatch and accent, and freshly reshaped glyphs, so every variation reads as a different plate.
  • Rougher + / Cleaner − — or the ↑ / ↓ keys — scale every perturbation from a clean plot up toward heavily hand-disturbed. The CLEAN … ROUGH meter shows where the human scale sits between 0 and 2.
  • Reset — returns the plate to its canonical state (variation 0, human x1.00), always one click away.
  • Download SVG / JSON / HPGL — takes the exact plate on screen to a plotter or vector tool. Seed, variation and human scale are printed on the plate, so any export is reproducible.
Core snippet
const penMap = { frame: 1, type: 2, hatch: 3, registration: 1 };
const ink = { 1: "#151515", 2: "#b5362b", 3: "#244f73" };

// Plate geometry derived once. Rows are offsets from INNER.y, so the whole plate
// rescales as one unit (INNER = 118, 118, 324, 324; ROW.hero = INNER.y + 70).
const FRAME = { x: 78, y: 78, size: 404 };
const INNER = { x: FRAME.x + 40, y: FRAME.y + 40, w: FRAME.size - 80, h: FRAME.size - 80 };
const ROW = { hero: INNER.y + 70, sub: INNER.y + 126, band: INNER.y + 182 };

// The composition as data: [layer, method, geometry, perturbation]. The stroke
// is derived from the pen map, so the array describes the plate, not the pen. A
// reroll advances `variation`, and composition(variation) walks small tables so
// each plate gets a new symbol set, a re-angled cross-hatch and a re-angled
// mid-tone accent -- the plate shifts, not just its glyphs.
const shapesFor = (comp) => [
  ["frame", "rect", [FRAME.x, FRAME.y, FRAME.size, FRAME.size], { hesitate: 0.35 }],
  ["type",  "text", ["PLOT", INNER.x, ROW.hero], { size: 88, glyphJitter: 0.7 }],
  ["hatch", "rect", [ACCENT.x, ACCENT.y, ACCENT.w, ACCENT.h], { fill: "hatch", hatchAngle: comp.accentAngle, alpha: 0.5 }],
  ["hatch", "symbols", [INNER.x, ROW.band, 174, 130], { set: comp.symbolSet, cluster: true, size: 13 }],
  ["hatch", "rect", [312, ROW.band, 130, 130], { fill: "cross", hatchAngle: comp.hatchAngle, hatchSpacing: comp.hatchSpacing }]
  // ...letters field + registration marks, same shape
];

// buildPlot() iterates the array; humanize() scales wobble/drift/... by a live
// factor so a button can slide the plate from machine-clean to hand-disturbed.
for (const [layer, method, geom, opts] of shapesFor(composition(variation))) {
  plot[method](...geom, { layer, stroke: ink[penMap[layer]], ...humanize(opts) });
}

const hpgl = plot.exportHPGL({ page, penMap, speed: 20 });

const stats = plot.stats({ page, drawSpeed: 20, travelSpeed: 60 });
// per pen: paths and work time -> stats.layers.type.drawnLength / drawSpeed
// whole plate: stats.estimatedSeconds -> total plot time
