Quickstart: chant()

Load the text addon and one verb performs the whole permutation poem: the phrase says its orders, each line cut a little deeper. This snippet is the entire sketch. Next: quickstart-plotter.

The whole sketch
// quickstart-chant - one phrase set free to say its orders.
function setup() {
  const canvas = createCanvas(560, 560);
  canvas.parent("sketch");
  pixelDensity(1);
  noLoop();

  const plot = new GysinPlot({ seed: 1960 });
  plot.chant("SAY IT AGAIN", 60, 110, { size: 34 });
  plot.draw();
}
This page loads p5.gysin.text.min.js after the core - that is the only extra step. The permutation_poem example is the same method, composed into a poster.
