underwood()

Typewriter

underwood() is an intent verb: defaults alone give a period-correct typewriter page. Single-stroke Hershey letterforms (bundled, no font to load), fixed monospace pitch and line height (10 chars-per-inch / 6 lines-per-inch, the classic Pica machine), and the small strike wear of a real typed sheet. Every option is opt-in, so plot.underwood("RUB OUT THE WORD", 60, 90) is a complete gesture.

A typewriter had no bold and no italic (see the decoration research). So the only decoration is what the machine could do: overstrike (bold: true, the same key struck twice), underline (the underscore rule), ALL CAPS, spacing, and typed symbol dividers. Mass comes from double-striking, never from fill.

Core snippet
// Load the underwood module, then the whole sketch is one verb:
const plot = new GysinPlot({ seed: 1961 });
plot.underwood("RUB OUT THE WORD", 60, 90);   // period typewriter, defaults
plot.draw();

// Period-correct decoration, all opt-in. No bold or italic ever existed
// on a typewriter, so "bold" is just the same key struck twice:
plot.underwood("HEADING", 60, 90, { size: 24, underline: 1 });
plot.underwood("DOUBLE-STRUCK", 60, 130, { bold: true });
plot.underwood("* * * * * * * *", 60, 170);   // a divider it could type

// wear scales the whole strike irregularity from one knob (0 = clean):
plot.underwood("RUB OUT THE WORD", 60, 210, { wear: 0 });

Sperrsatz

William Burroughs cut up the world on an Underwood No. 5, the flat black office standard that fixed the look of a typed page for half a century. Its keys carried one face, at one size, upright. There was no bold and no italic to reach for, because a typebar can only strike the single shape cast on it.

So emphasis was mechanical, not typographic. A word was made to matter by striking it twice (the nearest thing to bold), by underlining it with a run of underscores, by setting it in CAPITALS, or by Sperrsatz - spacing the letters apart so the word opened out along the line. Rows of asterisks, hyphens and underscores ruled the page into sections; tone came from overprinting characters, never from heavier ink. What a typewriter could not do - a second weight, a slant, another size, a curve - simply did not happen on the page.

underwood() stays inside those limits on purpose: one single-stroke face at one size, mass only from the double strike, decoration only from what the machine could actually type.
