Plotter Calibration

A practical A4 calibration sheet: margins, circles, squares, overshoot, dropout and three pens. Export it before a real plotting session and inspect its path statistics.

Core snippet
const page = {
  width: 210,
  height: 297,
  units: "mm",
  margin: 10,
  scale: 1 / 3,
  clip: true
};

plot.downloadHPGL("calibration.hpgl", {
  page,
  penMap: { frame: 1, circle: 2, hatch: 3 },
  speed: 20
});

const stats = plot.stats({ page, drawSpeed: 20 });
Ready: 10 mm margins, three pen layers.
