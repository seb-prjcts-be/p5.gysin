Plotter Calibration

Half of the Transmission room. The other half is Plotter Export: the same traces leaving the browser as SVG, JSON, and HPGL.

A readable A4 calibration instrument, plotted the way you actually plot - with just three pens. Black carries all the structure, text and measurement; a single red pen is reserved for the rings, so they read as the focal point rather than one accent among many; and a muted tint lays down the hatch-fill mass. That restraint gives the plate a real pen hierarchy and makes the HPGL export honest. The title is drawn through the composition layer - per-glyph jitter and a doubled stroke - so the header itself demonstrates the disturbance the sheet measures, with a small plot.letters() specimen beside it. Below it, every zone isolates one parameter: a mm ruler with numbered ticks; concentric rings labelled by diameter with a centre registration cross and a cross-hatched core, flanked by a pressure-toned cross-hatched ink mass and a dense field of true asemic marks from plot.asemic(); a mirrored ramp where dropout climbs 0.0 → 0.4 on the left and breathe climbs 0.0 → the slider value on the right; a hatch-density gradient of five diamonds, spacing 0.8 → 2.4 mm with a sweeping angle; a pen-weight ramp of bars with rising strokeWeight; and four segments isolating hesitate, fray, pressure and rubout, each above a straight shadow line so the effect reads against a true zero, closed by a full-width plot.symbols() band. Captions are kept small and light on purpose, so the rings and ink mass carry the image. All text is the built-in single-stroke alphabet, so the sheet stays plotter-safe.

How to read it: drag the Live breathe slider and watch the right-hand ramp - the zero line stays put at the top and fans out further as breathe rises, so you can see what the parameter does. The note under the canvas shows the live path cost (seed · paths · mm drawn · mm travel), and the legend maps each pen colour to its physical pen number for the HPGL export. Hit Reroll seed to jump the seed and watch that cost change, Reference sheet to snap back to the reproducible plate, then export to SVG or HPGL. The buttons are mirrored on the keyboard: 0, R, S, H.

Core snippet
// The whole core, in three lines - this alone is a complete sketch:
const plot = new GysinPlot({ seed: 210297 });
plot.text("P5.GYSIN CALIBRATION", 48, 39);   // clean, mechanical text
plot.draw();

// Everything below is OPTIONAL. This sheet is a physical instrument on an
// A4 page, so it works in millimetres and maps each layer to a pen:
const mm = (v) => v * MM;   // millimetres -> canvas pixels

// 1) a mm ruler: one tick plus its number, both single-stroke plotter-safe
plot.line(x, y, x, y - mm(5), { layer: "label" });
plot.text("50", x, y + mm(6), { size: mm(3), layer: "label" });

// 2) rings over a cross-hatched ink mass; pressure reads as tonal ink
plot.circle(cx, cy, d, { fill: "cross", hatchSpacing: mm(0.9), pressure: 0.4, layer: "fill" });

// 3) a type specimen: the built-in alphabet dissolved into varied glyphs
plot.letters("P5 GYSIN", x, y, w, h, { glyphJitter: 0.6, layer: "label" });

// …then dropout/breathe, hatch and pen-weight ramps, and disturbance verbs.

// Path cost shown up front, before any SVG or HPGL export:
const s = plot.stats({ page: PAGE, drawSpeed: 20, travelSpeed: 60 });
