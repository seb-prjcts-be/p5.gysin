textCutup() + vector traces

Gysin Demo

Cut-up typography, repeated contours, rubout gaps, uncertain plotter lines, and exportable vector output. Press S for SVG, J for JSON, or R for a new seeded variation.

Core snippet
const plot = new GysinPlot({ seed: 19600319 });

plot.textCutup("I THINK THEREFORE I AM", 76, 370, {
  size: 55,
  slices: 9,
  sliceOffset: 24,
  dropout: 0.09,
  repeat: 2,
  rubout: 0.12
});

plot.draw();
Keyboard: S SVG, J JSON, R new variation.
