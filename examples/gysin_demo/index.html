textCutup() + hatch fill + letter field

Gysin Demo

A cut-up typographic poster on the theme "rub out the word". The title comes apart letter by letter (glyphJitter), the lower shapes gain tonal mass through parallel and cross hatching (fill), and the word itself dissolves into a decaying letter field (letters()). Everything exports as clean plotter vectors.

The reference: "Rub out the word" is Brion Gysin's cut-up battle cry - the artist who, with William Burroughs, sliced text apart to break language open. This poster performs the phrase: it rubs itself out. Seed 19600319 fixes the noise, so every reload is the same print.

What to look for: follow the word down the page in zones - the rubbed-out titles, the ruled band that fans and tapers, the airy tilted slab weighed against the dense dark disc, and at the foot the word breaking apart into a decaying letter field. The rub-out is a gradient: light at the titles, heaviest at the field. Drag rub out to deepen that gradient top to bottom; drag break apart to shake every letter off its axis; hit New variation to re-decay everything except the frozen top title.

Core snippet
// The whole library, in three lines - this alone is a complete sketch:
const plot = new GysinPlot({ seed: 19600319 });
plot.text("RUB OUT", 75, 145);   // clean, mechanical text
plot.draw();

// Everything below is OPTIONAL disturbance layered on top. Defaults are all
// zero, so options are opt-in. This example adds, in order:

// 1) the two titles, rubbed away letter by letter (rubout + glyphJitter)
titleId = plot.text("RUB OUT",  75, 145, { size: 96, rubout: ruboutAt(145), glyphJitter: letterJitter });
plot.text("THE WORD", 76, 235, { size: 96, rubout: ruboutAt(235), glyphJitter: letterJitter });

// 2) a cut-up quotation, sliced and re-offset so the phrase stutters
plot.textCutup("I THINK THEREFORE I AM", 76, 370, { size: 55, slices: 9, sliceOffset: 24 });

// 3) a ruled band of six lines that fan out and taper away
for (let i = 0; i < 6; i++) plot.line(80 + i * 8, 448 + i * 18, 820, 448 + i * 23, { dropout: 0.04 + i * 0.03 });

// 4) two hatched masses in balance - both feel the break-apart slider too
plot.polygon(slab,         { fill: "hatch", hatchSpacing: 8, wobble: 1.3 + letterJitter * 1.2 });
plot.circle(705, 672, 160, { fill: "cross", hatchSpacing: 6, fray:  0.25 + letterJitter * 0.4 });

// 5) the word made literal, dropped to the foot as a decaying letter field
plot.letters("RUB OUT THE WORD", 90, 795, 720, 105, { size: 26, rubout: ruboutAt(795) });

plot.freeze(titleId); // reroll leaves the top title untouched
plot.draw();

now driving the code above: rubout 0.14 · glyphJitter 0.40

  2. Ruled band - liniatuur that fans out and tapers away
  3. Two masses - an airy hatch slab weighed against a dense cross-hatched disc
  4. Letter field - the word decaying into loose glyphs

Reading top to bottom, the word dissolves through all four zones.
