p5 Editor Starter

A paste-ready starter for the p5.js Web Editor. The plate acts out its title: THE MACHINE REMEMBERS is sliced once, then the word REMEMBERS returns lower and forgets itself - each lost letter decaying through its own shape shaken loose, then a printed sign, then a pure scribble. Raise decay and the ruled anchor and the machine-grid on the right erode too, so the forgetting reaches the whole plate. Everything on the canvas is real vector, so what you see is exactly what the SVG plots. The Editor's index.html already loads its own p5.js and sketch.js - make its script list match the three lines below exactly. Replace the Editor's existing p5.js line rather than adding a second one: two p5 copies on one page collide (_runLifecycleHook is not a function). Then paste the compact core below into sketch.js - it runs as-is. The full version of this page, with the buttons, keys and letter-by-letter forgetting, is sketch.js; it pastes just as cleanly.

// The whole library, in three lines - inside setup(), this alone is a plate:
//     const plot = new GysinPlot({ seed: 19600319 });
//     plot.text("CUT UP", 82, 150);   // clean, mechanical text
//     plot.draw();
// Everything else is OPTIONAL disturbance layered on top; defaults are all
// zero. This compact core stacks the same six layers as the full starter,
// back to front. Paste it into sketch.js as-is - it runs.

const decay = 0.45;     // how far the machine forgets; 0 keeps every letter
const INK = "#151515";  // machine grey
const RED = "#c0392b";  // second pen: the returning word and its ring
let plot;

function setup() {
  createCanvas(720, 720);
  noLoop();
  plot = new GysinPlot({
    seed: 19600319,
    width,
    height,
    style: { stroke: INK }
  });

  // 1) anchor - a ruled block (frame + closely set rules) grounds the plate
  plot.rect(88, 470, 544, 160, {
    fray: 0.3,
    dropout: 0.05 + decay * 0.2
  });
  for (let i = 1; i < 13; i++) {
    plot.line(88, 470 + i * 12.3, 632, 470 + i * 12.3, {
      dropout: 0.05 + decay * 0.35,
      rubout: decay * 0.22
    });
  }

  // 2) right-hand field - a light machine index (symbols over a grid)
  plot.symbols(470, 274, 196, 186, {
    size: 15,
    alpha: 0.35,
    dropout: 0.05 + decay * 0.5
  });
  plot.grid(470, 274, 196, 186, 3, 3, {
    frame: { stroke: INK }
  });

  // 3) red ring - circled around the returning word
  plot.circle(256, 337, 350, {
    stroke: RED,
    repeat: 3,
    rubout: 0.1,
    alpha: 0.7
  });

  // 4) title - the largest surface; hesitate breaks it up
  plot.text("CUT UP", 82, 150, {
    size: 96,
    hesitate: 0.5,
    rubout: 0.06
  });

  // 5) phrase - sliced once
  plot.textCutup("THE MACHINE REMEMBERS", 82, 262, {
    size: 46,
    slices: 8,
    rubout: 0.14
  });

  // 6) the word returns lower, red, already fraying - the full starter
  //    below forgets it letter by letter as decay rises
  plot.textCutup("REMEMBERS", 82, 360, {
    size: 46,
    stroke: RED,
    rubout: 0.14 + decay * 0.3
  });
}

function draw() {
  background("#f0efe9");
  plot.draw();
}

Watch: press Decay + and see REMEMBERS forget its last letters - each drops from its own shape shaken loose to a printed sign to a pure scribble, and the ruled anchor and right-hand grid erode with it. Decay − brings it back; Decay reset jumps to the 0.45 baseline, where several letters have already gone. The buttons show the live value and the stats line reads all three steerable axes - seed, hand and decay - plus how many of the nine letters went. Reseed redraws a visibly different work each time; Hand toggles the marks between steady and nervous. Nothing here is screen-only decoration, so Download SVG plots exactly what you see. The buttons match the R, H, + / −, 0 and S keys.
