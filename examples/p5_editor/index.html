p5 Editor Starter

A paste-ready starter for the p5.js Web Editor. The plate acts out its title: THE MACHINE REMEMBERS is sliced once, then the word REMEMBERS returns lower and forgets itself — each lost letter decaying through its own shape shaken loose, then a printed sign, then a pure scribble. Raise decay and the hatched anchor and the machine-grid on the right erode too, so the forgetting reaches the whole plate. Everything on the canvas is real vector, so what you see is exactly what the SVG plots. Add the CDN script in index.html, then paste the sketch into sketch.js.

sketch.js core
// The whole library, in three lines — this alone is a complete sketch:
const plot = new GysinPlot({ seed: 19600319 });
plot.text("CUT UP", 82, 150);   // clean, mechanical text
plot.draw();

// Everything below is OPTIONAL disturbance layered on top. Defaults are all
// zero, so options are opt-in. buildPlot() stacks six layers, back to front:

// 1) anchor — a cross-hatch-filled block that grounds the plate
plot.rect(88, 470, 544, 160, { fill: "cross", dropout: 0.05 + decay * 0.35 });

// 2) right-hand field — a light machine index (symbols over a grid)
plot.symbols(470, 274, 196, 186, { size: 15, dropout: 0.05 + decay * 0.5 });
plot.grid(470, 274, 196, 186, 3, 3, { frame: { stroke: INK } });

// 3) red ring — circled around the returning word
plot.circle(ringX, ringY, 350, { stroke: RED, repeat: 3, rubout: 0.1 });

// 4) title — the largest surface; hesitate breaks it up
plot.text("CUT UP", 82, 150, { size: 96, hesitate: 0.5, rubout: 0.06 });

// 5) sentence — sliced once by the seed's grain
plot.textCutup("THE MACHINE REMEMBERS", 82, 262, { slices: 8, rubout: 0.14 });

// 6) the word returns lower and forgets itself, letter by letter —
//    each lost glyph decays: its own shape shaken loose (letters) →
//    a printed sign (symbols) → a pure scribble (asemic)
degradeWord("REMEMBERS", 82, 360, 46, decay, hand);

plot.draw();

Watch: press Decay + and see REMEMBERS forget its last letters — each drops from its own shape shaken loose to a printed sign to a pure scribble, and the hatched anchor and right-hand grid erode with it. Decay − brings it back; Decay reset jumps to the 0.45 baseline, where several letters have already gone. The buttons show the live value and the stats line reads all three steerable axes — seed, hand and decay — plus how many of the nine letters went. Reseed redraws a visibly different work each time; Hand toggles the marks between steady and nervous. Nothing here is screen-only decoration, so Download SVG plots exactly what you see. The buttons match the R, H, + / −, 0 and S keys.
