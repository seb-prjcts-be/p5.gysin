Signal Score

Five paths read as fragile signals, each with its own waveform and voice, labelled S1–S5 in the margin. Every signal is marked at its loudest point by a diamond sized to - and now printed with - that loudness as a percent, and a thin thread drops from each diamond to the asemic notation cell that transcribes it. The red row (S3) leads the five, anchored by a thin red baseline. A small legend in the bottom margin names the three marks, so you can decode the score straight from the image. Click the composition or press R for a new reading; the wave forms reshape while the row structure stays put.

Core snippet
// The whole library, in three lines - this alone is a complete sketch:
const plot = new GysinPlot({ seed: 40017 });
plot.path([[52, 200], [668, 200]], { stroke: "#171717" });   // one clean signal
plot.draw();

// Everything below is OPTIONAL, layered on top. Defaults are all zero, so
// options are opt-in. This score adds, back to front:

// 1) five signal paths - the accent row (S3) red and heavy, the rest lighter
plot.path(signal, {
  stroke: accent ? "#b5362b" : "#171717",
  strokeWeight: accent ? 2 : 1, alpha: accent ? 0.9 : 0.6, layer: "signal"
});

// 2) a row of asemic notation, taller and denser where the score is louder
plot.asemic(x, top, w, h, { loops: 3 + round(amp * 6), layer: "notation" });

// 3) each signal's peak marked by a hatch-filled diamond sized to its
//    loudness, and threaded down to the notation cell that transcribes it
plot.polygon(diamond(px, py, r), { fill: "hatch", layer: "markers" });

plot.draw();

Each reading has a reproducible seed, path count and reading-head channel.

Start with the red row (S3): it runs unbroken from signal to marker to notation while the four ink voices sit back as backdrop. Then read the rest - each line S1–S5 is a signal, darker and heavier the louder its voice; each diamond is a loudest-point marker, sized and labelled with its loudness percent; a thread ties it to the asemic notation cell that transcribes it. The lone faint vertical is the reading head on the loudest column, tagged with its level and named in the caption above. New reading reshapes the wave forms; Download SVG exports the plot. R and S do the same from the keyboard.
