Signal Score

No letters: this score treats paths as fragile signals. Click the composition or press R to generate a new machine reading while keeping the same score structure.

Core snippet
const signal = [];
for (let x = 0; x <= 620; x += 8) {
  signal.push([x, y + sin(x * 0.04) * 28]);
}

plot.path(signal, {
  wobble: 1.2,
  dropout: 0.07,
  repeat: 2,
  rubout: 0.16,
  layer: "signal"
});

plot.polygon(nodes, { layer: "markers" });
Click the score or press R for a new variation. Press S to export SVG.
