Signal Score

Five paths read as fragile signals, each with its own waveform and voice, labelled S1–S5 in the margin. Every signal is marked at its loudest point by a diamond sized to — and now printed with — that loudness as a percent, and a thin thread drops from each diamond to the asemic notation cell that transcribes it. The red row (S3) leads the five, anchored by a red hatch band and baseline. A small legend in the bottom margin names the three marks, so you can decode the score straight from the image. Click the composition or press R for a new reading; the wave forms reshape while the row structure stays put.

Core snippet
// hero anchor: a faint red hatch fills the band between the accent signal and
// its baseline, giving S3 one clear focus (polygon has a fillable interior)
plot.polygon(heroBand, {
  fill: "hatch", hatchSpacing: 3, stroke: "#b5362b", alpha: 0.15, layer: "signal"
});

// louder voices sit darker and heavier, quieter ones recede -> depth
plot.path(signal, {
  stroke: accent ? "#b5362b" : "#171717",
  strokeWeight: accent ? 2 : map(v.amp, 12, 30, 0.6, 1.4, true),
  alpha: accent ? 0.9 : map(v.amp, 12, 30, 0.5, 0.85, true), layer: "signal"
});

// the notation cell that receives S3's thread is inked red, so the accent runs
// unbroken from signal to marker to notation
plot.asemic(x, top, w, h, {
  strokeWeight: 0.6 + amp * 1.4 + (linked ? 0.5 : 0),
  stroke: i === accentCell ? "#b5362b" : "#3a3a3a", layer: "notation"
});

Each reading has a reproducible seed, path count and reading-head channel.

Start with the red row (S3): it runs unbroken from signal to marker to notation while the four ink voices sit back as backdrop. Then read the rest — each line S1–S5 is a signal, darker and heavier the louder its voice; each diamond is a loudest-point marker, sized and labelled with its loudness percent; a thread ties it to the asemic notation cell that transcribes it. The lone faint vertical is the reading head on the loudest column, tagged with its level and named in the caption above. New reading reshapes the wave forms; Download SVG exports the plot. R and S do the same from the keyboard.
