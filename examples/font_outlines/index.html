p5.Font · contours · counters

Font Outlines

Load a real font, then let p5.gysin keep the outer contours and inner counters as separate plotter paths. The title is deliberately made of letters with holes.

Core snippet
const outlineFont = await loadFont(FONT_URL);

plot.text("O8BR", 70, 190, {
  font: outlineFont,
  size: 150,
  wobble: 0.9,
  repeat: 2,
  rubout: 0.08
});

plot.textCutup("COUNTER MEMORY", 72, 380, {
  font: outlineFont,
  size: 48,
  slices: 8,
  sliceOffset: 15
});
